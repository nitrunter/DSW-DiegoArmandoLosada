Tarea 3: Git y Github

El presente ovi tiene como finalidad, enseñar los conceptos báicos de git y git hub.
Deberá navegar por las diferentes opciones del menú desplegable para acceder a la
información contenida en el ovi.



Objetivo general

Diseñar y desarrollar el OVA y OVI para dar solucion al tema 2 del curso de diseño web, el tema seleccionado por mi es

Git y GitHub, tema con el que desarrollaré el presente taller


Objetivos especificos


    • Desarrollar un ovi de aprendizaje, como tema de git, incluyendo uso, definicion y dominio del tema

    • Diseñar un material de estudio tipo Ovi, con el que se enseñ el uso de Github, tanto en aspecto teorico como practico

  • Dar solucion al taller 2 de diseños de sitios web de la UNAD, con el tema de git y github que selecciono como el estudiante 1 en el foro
